核心功能

像用 Figma 一样
改线上页面

完整的样式调整面板，覆盖前端日常调整的所有属性。

元素精准选取

鼠标悬停绿色虚线高亮，点击选中并自动生成 CSS 选择器，过滤随机 class，确保选择器稳定可复用。

实时样式编辑

排版、尺寸、间距、外观、边框——所有属性即时预览，改完立即看到效果，无需刷新。

间距可视化

选中元素时，margin 橙色、padding 青色区域直接标在页面上，像素值一目了然。

图片替换

本地上传图片直接替换页面中的 img 或背景图，所见即所得验证图片效果。

系统字体读取

通过 Local Font Access API 读取系统全部字体，字体选择和实际开发环境完全一致。

一键导出指令

所有修改自动生成为结构化 Prompt，包括文字内容、CSS 属性、图片替换，粘贴到 Trae / Cursor 等 AI 编码工具即可自动改代码。

Margin & Padding 一眼看清

选中元素后，页面直接渲染出 margin（橙色）和 padding（青色）区域，每个区域标注像素值。修改间距数值时，可视化实时更新。

智能选择器生成

自动过滤 Tailwind 随机 class、动态属性，生成稳定可用的 CSS 选择器。支持 tag + class 组合，确保导出指令中的选择器在你的代码库中能精确定位。

nav .header-nav
height: 64px → 72px
padding: 0 24px → 0 32px
a.nav-link
font-size: 14px → 15px
letter-spacing: -0.18px → -0.2px
导出指令
1. nav.header-nav
- height: 64px → 72px
- padding: 0 24px → 0 32px
2. a.nav-link
- font-size: 14px → 15px
- letter-spacing: -0.18px → -0.2px
已复制到剪贴板

一键导出，粘贴即用

所有修改自动生成为结构化 Prompt，支持文字内容、图片替换、CSS 属性变更。直接粘贴到 Trae、Cursor、Claude 等 AI 编码工具，自动完成代码修改。
